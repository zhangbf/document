1.使用myeclipse的版本为10.5， 对应version 为3.7.2， Git 2.2.0的下载可参考 http://download.csdn.NET/detail/p1i2n3g4/9703197； 2.将插件下载完成后，在myeclipse安装目录dropins下新建egit文件夹，将下载文件全部复制进去。完成以后，重启myeclipse，即可在 windows --show view中看到git Repositories；git插件集成完毕； 3.使用的代码托管平台为 oschina的 码云，首先注册oschina，码云的账户， 4.oschina码云中新建项目，可以是private，或者public，public比较简单，这里介绍下private模式需要的设置；输入name,description,language,gitlgnore,select private isnot;click new button done;   4.1码云平台点击右上角头像，页面坐下看到Repositories可以点选public或者private查看公开或未公开的项目，（因语言为英文，所以有所区别）；   4.2因需要将本地项目放到云上，所以去掉 readme文档的选择，默认选择 readme。 5.将本地project add 码云，首先在本地建立git 仓库，将project放到git 仓库中。   5.1建立本地git仓库：myeclipse中打开 git Repositores面板，点击右侧小图标（create a new git repositores add it to this view），选择一个路径，可以新建，然后填上name，finish；   5.2在myeclipse  package explore面板中右键项目，team---》shar project---》选择git---》next--》在 Repository处 选择刚才创建的git 仓库路径，点击finish完成；   5.3此刻项目上将出现 ？的小图标，右键项目--team---commit---》输入 commit message内容， 在files的右侧点击全选小图标，全部选中，点击commit按钮提交到本地git仓库，也可点击commit and push 直接推送到码云的远程仓库；我们首先提交到本地git，然后在push到码云平台；   5.4commit完成之后，可以在 git repositories中---》branches---》local目录下看到新增加了一个主分支 master；表示项目已添加到本地git  master主分支下；下面我们将master push 到我们刚才新建的码云项目中去；（切记码云中新建的项目中不能有readme文件，不然有冲突）；   5.5在我们的本地git仓库中右键----》push---》URI：为我们在码云上项目的路径，（在码云上，进入刚才的项目，可以在右侧看到路径，一个HTTPS,一个SSH，我们把https的路径复制下来）填入URI，host和 repository path会自动生成，connction不用考虑，user和pwd中，输入码云上的用户名密码，点击next；   5.6在 source ref： 下 点击下拉箭头，选择refs/head/master，点击右侧的 Add Spec,添加到 Specifications for push中去，然后点击finish按钮完成操作。 备注：push的时候，需要git的用户名，密码，这些信息是之前下载的git的 exe文件，安装完成后，通过 git  global。。。的设置的用户名，邮箱，可以在网上搜一下很多这样的介绍。Git-2.11.0-64-bit.exe，安装的这个，然后在base 命令行中配置用户名和邮箱什么的，就可以了。 6.从码云上克隆项目到本地git仓库：在myeclipse中Git Respositories面板右侧，点击clone a git  respositores  and the clone to this view小图标，在URI中输入需要clone项目的地址，输入user ，pwd ，完成；（如果项目为私有未公开的，则需要管理员在项目中点击setting --->Members settings--->developer--->右上角add member-->输入开放给某个用户的用户名，点击ADD USERS TO..）完成操作。 7.从本地git仓库将项目添加到myeclipse开发：在新导入的本地git上右键，import project--》next----》finish； 欢迎指正；
